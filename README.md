# Aspis Environmental – Presentation / Demo

**Aspis Environmental** is a specialized lead-based paint inspection service that brings **verification, tracking, and transparency** to counties and municipalities.

While most inspection providers only issue a report, Aspis is the only provider that gives counties **real-time visibility**, tamper-resistant verification, and full audit trails for every inspection.

### Core Pillars
- **Inspection** – Professional lead-based paint inspections (visual + dust wipe sampling)
- **Verification** – Unique report IDs with public online validation
- **Transparency** – Real-time dashboard for county admins with filtering, statistics, and export

### Key Features
- Role-based access (Inspectors, Realtors, County Admins)
- Real-time tracking dashboard with statistical process control
- Public verification portal (anyone can validate a report using the tracking number)
- Multi-lingual interface
- CSV export and compliance reporting
- Online booking & payment integration

- Dust Wipe Sampling: **$350**
